# Vestige

**Decision archaeology for your codebase.**

Vestige reads your Git history to surface the why behind every change — 
so you stop inheriting mystery code and start understanding the decisions 
that shaped it.

## What it does

- Analyzes Git history, diffs, commit messages, and PR context
- Infers the reason behind undocumented changes using pattern recognition
- Returns tiered confidence output — High, Medium, or Low — based on 
  available evidence
- Flags knowledge gaps where decisions were made and never recorded
